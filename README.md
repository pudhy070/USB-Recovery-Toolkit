## USB 복구 툴


📌 프로젝트 소개

USB 바이러스 복구 툴 v2.0은 Windows 환경에서 USB 드라이브 및 외장 저장 장치의 숨김, 시스템, 읽기 전용 속성을 제거하고, 의심스러운 파일을 검사 및 정리할 수 있는 GUI 기반 복구 도구입니다.
관리자 권한이 필요한 딥 스캔 기능과 휴지통 복구 기능도 제공합니다.

⚡ 주요 기능

- USB 드라이브 감지: 연결된 USB 및 외장 드라이브 자동 감지

- 폴더 선택: 복구할 대상 경로 선택

- 숨김 파일/폴더 복구: 숨김(H) 속성 제거

- 시스템 속성 제거: 시스템(S) 속성 제거

- 읽기 전용 속성 제거: 읽기 전용(R) 속성 제거

- 바이러스 검사 & 정리: 자동 생성 .lnk, autorun.inf, .vbs, .bat, .cmd 파일 탐지 및 정리

- 휴지통 복구: Windows 휴지통에서 삭제된 파일 확인 및 복구 유도

- 딥 스캔 복구: chkdsk를 활용한 파일 시스템 검사 및 손상 파일 복구

- 실시간 로그 출력: 진행 상황 및 작업 로그 확인 가능

- 진행률 표시: 복구 작업 진행 상태 표시

---

🖥️ 실행 환경

Windows 10 이상

Python 3.10 이상

필요 라이브러리

```
pip install psutil
```
🚀 사용 방법

저장소 클론 또는 다운로드
```
git clone https://github.com/pudhy070/USB-Recovery-Toolkit.git
cd USB-Recovery-Toolkit
```

Python 실행
```
python usb_tool.py
```

GUI에서 다음 단계 진행

USB 드라이브 또는 복구할 폴더 선택

복구 옵션 선택 (숨김, 시스템, 읽기 전용 속성 제거)

필요 시 바이러스 검사, 휴지통 복구, 딥 스캔 실행

로그 및 진행률 확인

⚠️ 주의 사항

딥 스캔 기능은 관리자 권한이 필요할 수 있으며, 실행 시 시간이 오래 걸릴 수 있습니다.

휴지통 복구 기능은 수동 복구 안내를 포함합니다.

Windows 전용 프로그램입니다.

중요 데이터는 반드시 백업 후 사용하세요.
